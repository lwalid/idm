To install UML Designer, it is possible to:

  • download a bundle, in this case UML designer is already installed, just unzip and run the umldesigner executable.
  • get UML Designer in an existing Eclipse from the marketplace or from an update-site. If you decide to install it like this, during the installation if you get the security warning saying that the authenticity or validity of the software can’t be established, just click OK.

Pay attention that the nightly build is unstable, this build is for testing purposes only.

Bundles

The bundles corresponding to the last release are available on the Obeo marketplace :

  • Windows 32bits
  • Windows 64bits
  • Linux 32bits
  • Linux 64bits
  • MacOs 64bits

Marketplace

It is also possible to install UML Designer from the Eclipse marketplace or the Obeo marketplace Obeo marketplace. Don’t know how to install a module from the marketplace ? Have a look to the marketplace client introduction.

Update Sites

The update-sites provide a p2 repository for UML Designer. Don’t know how to install a module from an Update Site ? Have a look to our video or to the full procedure description.
